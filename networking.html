Networking Concepts

DNS

DNS (Domain Name System) translates human readable domain names into IP addresses. For example, when a user visits a website, DNS resolves the domain name into the server's IP address.

IP Addressing

Every device connected to the internet has an IP address. Websites are hosted on servers with IPv4 or IPv6 addresses.

HTTP vs HTTPS

HTTP is the protocol used to transfer web pages. HTTPS is the secure version which encrypts communication using TLS.
